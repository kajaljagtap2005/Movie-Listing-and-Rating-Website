 The project displays movies with details like title, genre, release year, poster, and allows users to rate movies through an interactive star rating system.

 # CineMagic – Premium Movie Portal

A modern, beautiful movie browsing experience built with HTML, CSS, and pure JavaScript.  
Browse hand-picked movies, watch official trailers, rate films with interactive stars, and leave feedback — all wrapped in a sleek dark theme.

## Features

- Dark modern UI with gradient accents  
- Responsive movie grid layout (works great on mobile & desktop)  
- Real movie posters, titles and release years powered by TMDB  
- Hover + click interactive 5-star rating system  
- Watch official YouTube trailers in a smooth modal popup  
- Animated feedback form with success message  
- Smooth scroll animations (AOS library)  
- Clean, well-commented vanilla JavaScript code

## Built With

- HTML5  
- CSS3 (Custom properties, Flexbox, Grid, Gradients)  
- Vanilla JavaScript (ES6+, Fetch API, async/await)  
- [The Movie Database (TMDB)](https://www.themoviedb.org)  
- AOS – Animate On Scroll  
- Font Awesome  
- Google Fonts (Poppins)
